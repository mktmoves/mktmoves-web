# MKT MOVES

**Follow the Smart Money.**

MktMoves parses SEC filings into structured, searchable intelligence. Every quarter, big funds report exactly what they own. We make that data easy to read.

🌐 [mktmoves.com](https://mktmoves.com)

---

## What Is This?

MktMoves is a SaaS platform that transforms public SEC EDGAR filings into actionable analytics. Starting with 13F-HR filings (quarterly institutional holdings), with planned expansion to SC 13D/G (activist positions), Form 4 (insider trades), and S-1 (IPO activity).
